# 🪨 Классификатор типов руд

Автоматическая система классификации полированных шлифов руд на основе анализа минералогических признаков и пространственных отношений между фазами (Region Adjacency Graph).

## 🎯 Возможности

- **Автоматическая сегментация** изображения на три фазы: руда, тальк, пустая порода
- **Анализ морфологии зёрен**: размер, количество, распределение
- **Матрица смежности фаз**: анализ контактов между минералами
- **Классификация на 3 типа руд**:
  - 🟢 **Рядовая** — крупнозернистая, легко обогащается
  - 🔴 **Труднообогатимая** — мелкозернистая, сильно замещённая
  - 🔵 **Оталькованная** — руда окружена тальком
- **Интерактивный веб-интерфейс** с визуализацией результатов
- **Геологическое объяснение** каждого решения
