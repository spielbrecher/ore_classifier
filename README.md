# 🪨 Классификатор типов руд

Автоматическая система классификации полированных шлифов руд на основе анализа минералогических признаков и пространственных отношений между фазами (Region Adjacency Graph).

## 🎯 Возможности

- **Автоматическая сегментация** изображения на три фазы: руда, тальк, пустая порода
- **Анализ морфологии зёрен**: размер, количество, распределение
- **Матрица смежности фаз**: анализ контактов между минералами
- **Классификация на 3 типа руд**:
  - 🟢 **Рядовая** — крупнозернистая, легко обогащается
  - 🔴 **Труднообогатимая** — мелкозернистая, сильно замещённая
  - 🔵 **Оталькованная** — руда окружена тальком
- **Интерактивный веб-интерфейс** с визуализацией результатов
- **Геологическое объяснение** каждого решения

### 📊 Полное описание и интерпретация всех 32 признаков модели

Все признаки извлекаются из сегментированного изображения шлифа, где выделены три фазы:
- **Фаза 0** — тёмная (пустая порода, матрица)
- **Фаза 1** — серая (тальк + зоны замещения)
- **Фаза 2** — яркая (руда/сульфиды)

---

## 🎨 Группа 1: Фазовый состав (3 признака)

### `dark_area_pct` — доля матрицы
- **Формула:** `(пикселей_фазы_0 / всего_пикселей) × 100%`
- **Смысл:** Процент пустой породы в образце
- **Интерпретация:** Высокое значение означает бедную руду с большим содержанием нерудного материала
- **Типичные значения:** 40-70%

### `gray_area_pct` — доля серой фазы (тальк + замещение)
- **Формула:** `(пикселей_фазы_1 / всего_пикселей) × 100%`
- **Смысл:** Суммарная доля талька и зон замещения сульфидов
- **Интерпретация:** Ключевой индикатор оталькованности. Высокое значение → проблемы с флотацией
- **Важность:** Средняя (0.032)
- **Ожидаемые значения:**
  - 🟢 Рядовая: 10-17%
  - 🔴 Труднообогатимая: 15-20%
  - 🔵 Оталькованная: 15-25%

### `bright_area_pct` — доля руды ⭐
- **Формула:** `(пикселей_фазы_2 / всего_пикселей) × 100%`
- **Смысл:** Процент полезного минерала в образце
- **Интерпретация:** Главный показатель технологической ценности руды
- **Важность:** **0.105** (2-й по важности признак!)
- **Ожидаемые значения:**
  - 🟢 Рядовая: 20-75%
  - 🔴 Труднообогатимая: **2-21%** (критически мало!)
  - 🔵 Оталькованная: 13-66%

## 💡 Группа 2: Интенсивности фаз (6 признаков)

### `dark_mean_intensity` — средняя яркость матрицы
- **Формула:** `mean(gray[фаза_0])`
- **Смысл:** Средний уровень серого в тёмных областях (0-255)
- **Интерпретация:** Зависит от условий съёмки и минерального состава матрицы
- **Важность:** 0.028

### `gray_mean_intensity` — средняя яркость талька
- **Формула:** `mean(gray[фаза_1])`
- **Смысл:** Средняя яркость серой фазы
- **Интерпретация:** Тальк обычно имеет характерный "перламутровый" блеск со специфической яркостью
- **Важность:** 0.021

### `bright_mean_intensity` — средняя яркость руды
- **Формула:** `mean(gray[фаза_2])`
- **Смысл:** Средняя отражательная способность рудных минералов
- **Интерпретация:** Сульфиды (пирит, халькопирит) имеют высокую отражательную способность
- **Важность:** Средняя

### `dark_std_intensity` — неоднородность матрицы
- **Формула:** `std(gray[фаза_0])`
- **Смысл:** Насколько яркость матрицы варьируется
- **Интерпретация:** Высокое значение = разнородная матрица (несколько минералов)
- **Важность:** 0.022

### `gray_std_intensity` — неоднородность талька
- **Формула:** `std(gray[фаза_1])`
- **Смысл:** Вариативность яркости в серой фазе
- **Интерпретация:** Высокое значение указывает на смешение талька с другими минералами
- **Важность:** **0.037**

### `bright_std_intensity` — неоднородность руды ⭐
- **Формула:** `std(gray[фаза_2])`
- **Смысл:** Вариативность яркости внутри рудных зёрен
- **Интерпретация:** **Ключевой индикатор замещённости!** Высокое значение = внутри зёрен есть тёмные включения (нерудные минералы)
- **Важность:** **0.097** (3-й по важности признак!)
- **Ожидаемые значения:**
  - 🟢 Рядовая: низкая (однородные зёрна)
  - 🔴 Труднообогатимая: **высокая** (сильное замещение)

## 🔗 Группа 3: Матрица смежности фаз (9 признаков)

Матрица `adj_i_j` показывает **нормализованную долю контактов** между фазой `i` и фазой `j` среди всех границ на изображении.

            Фаза 0    Фаза 1    Фаза 2
Фаза 0      adj_0_0   adj_0_1   adj_0_2
Фаза 1      adj_1_0   adj_1_1   adj_1_2
Фаза 2      adj_2_0   adj_2_1   adj_2_2
```

### `adj_0_0` — контакты матрица↔матрица
- **Смысл:** Доля внутренних границ матрицы
- **Интерпретация:** Высокое значение = крупная непрерывная матрица

### `adj_0_1` — контакты матрица↔тальк
- **Смысл:** Как часто матрица граничит с тальком
- **Важность:** **0.130** (2-й по важности!)
- **Интерпретация:** Показывает степень "проникновения" талька в породу

### `adj_1_0` — контакты тальк↔матрица
- **Смысл:** Симметрично adj_0_1 (та же информация)
- **Важность:** 0.100

### `adj_0_2` — контакты матрица↔руда
- **Смысл:** Доля прямых контактов руды с пустой породой
- **Интерпретация:** Высокое значение = руда хорошо обособлена (хорошо для обогащения)
- **Важность:** 0.021

### `adj_2_0` — контакты руда↔матрица
- **Смысл:** Симметрично adj_0_2
- **Важность:** 0.021

### `adj_1_1` — контакты тальк↔тальк
- **Смысл:** Внутренние границы тальковой фазы
- **Интерпретация:** Высокое значение = крупные скопления талька

### `adj_1_2` — контакты тальк↔руда
- **Смысл:** Доля контактов между тальком и рудой
- **Важность:** 0.081

### `adj_2_1` — контакты руда↔тальк ⭐
- **Смысл:** Симметрично adj_1_2, но акцент на руду
- **Интерпретация:** **Критический индикатор!** Чем выше значение, тем хуже обогащение
- **Важность:** 0.070
- **Ожидаемые значения:**
  - 🔵 Оталькованная: очень высокое
  - 🟢 Рядовая: умеренное

### `adj_2_2` — контакты руда↔руда
- **Смысл:** Доля контактов рудных зёрен друг с другом
- **Интерпретация:** Высокое значение = крупные слипшиеся зёрна руды (хорошо для обогащения)
- **Важность:** 0.000 (не информативен в наших данных — руда редко контактирует сама с собой)

## ⛏️ Группа 4: Геологические признаки (5 признаков)

### `ore_talc_contact` — доля контактов руда↔тальк
- **Формула:** `adj_2_1 / (adj_2_0 + adj_2_1 + adj_2_2)`
- **Смысл:** Какая доля всех контактов руды приходится на тальк
- **Интерпретация:** Главный индикатор "обволакивания" руды тальком
- **Важность:** 0.028
- **Критическое значение:** >80% → оталькованная руда

### `ore_matrix_contact` — доля контактов руда↔матрица
- **Формула:** `adj_2_0 / (adj_2_0 + adj_2_1 + adj_2_2)`
- **Смысл:** Какая доля контактов руды приходится на пустую породу
- **Интерпретация:** Высокое значение = руда хорошо обособлена от талька
- **Важность:** 0.029

### `ore_ore_contact` — доля контактов руда↔руда
- **Формула:** `adj_2_2 / (adj_2_0 + adj_2_1 + adj_2_2)`
- **Смысл:** Какая доля контактов руды приходится на другие рудные зёрна
- **Интерпретация:** Индикатор крупных слипшихся зёрен
- **Важность:** 0.000 (в наших данных руда редко контактирует сама с собой)

### `ore_isolation_index` — индекс изоляции руды ⭐
- **Формула:** `ore_ore_contact - ore_talc_contact`
- **Смысл:** Разница между "хорошими" (руда-руда) и "плохими" (руда-тальк) контактами
- **Интерпретация:**
  - **Положительный** → руда изолирована (легко обогащается)
  - **Отрицательный** → руда окружена тальком (проблемы)
- **Важность:** 0.032
- **Типичные значения:** от -0.9 до +0.3

### `boundary_density` — плотность границ
- **Формула:** `total_boundary_pixels / total_pixels`
- **Смысл:** Общая длина всех межфазных границ на единицу площади
- **Интерпретация:** Высокое значение = тонкозернистая текстура, много мелких объектов
- **Важность:** 0.024
- **Ожидаемые значения:**
  - 🔴 Труднообогатимая: высокая (много мелких зёрен)
  - 🟢 Рядовая: умеренная

## 🔬 Группа 5: Морфологические признаки (9 признаков)

### `num_dark_regions` — число областей матрицы
- **Формула:** количество связных компонентов фазы 0
- **Смысл:** Сколько отдельных "кусков" пустой породы на срезе

### `num_gray_regions` — число областей талька
- **Формула:** количество связных компонентов фазы 1
- **Смысл:** Сколько отдельных скоплений талька

### `num_bright_regions` — число зёрен руды ⭐
- **Формула:** количество связных компонентов фазы 2
- **Смысл:** Сколько отдельных рудных зёрен на срезе
- **Важность:** 0.041
- **Интерпретация:** Большое число = мелкозернистая руда (плохо)
- **Ожидаемые значения:**
  - 🔴 Труднообогатимая: **1100-5900** (критически много!)
  - 🟢 Рядовая: 700-2900
  - 🔵 Оталькованная: 365-2550

### `avg_dark_area` — средний размер областей матрицы
- **Формула:** `(пикселей_фазы_0) / num_dark_regions`
- **Смысл:** Средняя площадь одного "куска" матрицы

### `avg_gray_area` — средний размер областей талька
- **Формула:** `(пикселей_фазы_1) / num_gray_regions`
- **Смысл:** Средняя площадь одного скопления талька

### `avg_bright_area` — средний размер зерна руды ⭐
- **Формула:** `(пикселей_фазы_2) / num_bright_regions`
- **Смысл:** Средняя площадь одного рудного зерна
- **Важность:** **0.113** (САМЫЙ ВАЖНЫЙ ПРИЗНАК!)
- **Интерпретация:** Чем больше зёрна, тем легче их выделить при обогащении
- **Ожидаемые значения:**
  - 🟢 Рядовая: крупные зёрна
  - 🔴 Труднообогатимая: **очень мелкие зёрна**
  - 🔵 Оталькованная: средние

### `max_bright_area` — размер крупнейшего зерна руды ⭐
- **Формула:** максимальная площадь среди всех рудных зёрен
- **Смысл:** Размер самого большого рудного зерна на срезе
- **Важность:** **0.088** (4-й по важности признак!)
- **Интерпретация:** Крупные зёрна = хорошие шансы на раскрытие при дроблении

### `bright_area_cv` — коэффициент вариации размеров зёрен
- **Формула:** `std(площадей_зёрен) / mean(площадей_зёрен)`
- **Смысл:** Насколько однородны размеры рудных зёрен
- **Интерпретация:**
  - Низкое значение = все зёрна одного размера (хорошо для обогащения)
  - Высокое значение = широкий разброс (сложнее настраивать процесс)
- **Важность:** 0.048

### `ore_grain_density` — плотность зёрен руды
- **Формула:** `num_bright_regions / (total_pixels / 10000)`
- **Смысл:** Количество рудных зёрен на 10000 пикселей
- **Интерпретация:** Нормализованный показатель зернистости
- **Важность:** 0.034
- **Ожидаемые значения:**
  - 🔴 Труднообогатимая: очень высокая плотность
  - 🟢 Рядовая: умеренная

## 🎯 Ключевые геологические инсайты

### Что модель "поняла" о рудах:

**1. Размер зёрен важнее всего**
Четыре из топ-5 признаков — про морфологию руды. Геологическая истина: чем крупнее зёрна, тем легче их отделить при обогащении.

**2. Неоднородность яркости = замещённость**
Признак `bright_std_intensity` ловит тонкие срастания: когда внутри светлого зерна руды есть тёмные включения нерудных минералов.

**3. Пространственные отношения важнее абсолютных значений**
Признаки из матрицы смежности (`adj_*`) дают информацию о том, КАК фазы расположены друг относительно друга — это то, что видит геолог, рассматривая шлиф.

**4. Комбинация признаков работает**
Модель использует не один "серебряный пуля"-признак, а комплекс из ~10 информативных метрик, что делает её устойчивой к вариациям данных.

## 🧪 Как использовать эти признаки на практике

### Для геологов:
- **Смотрите на `avg_bright_area` и `num_bright_regions`** — они говорят о текстуре руды
- **Следите за `ore_talc_contact`** — если >80%, ждите проблем с флотацией
- **Анализируйте `bright_std_intensity`** — высокий = тонкие срастания

### Для инженеров-обогатителей:
- **`bright_area_pct`** → ожидаемое извлечение
- **`avg_bright_area`** → необходимый помол
- **`ore_isolation_index`** → выбор реагентов

### Для data scientists:
- **Топ-10 признаков** можно использовать для упрощения модели
- **Признаки 2-5 и 9-10** хорошо коррелируют — можно делать feature selection
- **Признаки adj_2_2 и ore_ore_contact** неинформативны в наших данных → можно удалить
